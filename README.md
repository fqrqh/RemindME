# REMIND ME - Remind Yourself!

RemindME is a Small little program written in python that Let you choose How and When you can be reminded

for example, you need to work on somethings ? use RemindME to dont miss it!!

## ❓Usage

In **Config.py** you can Customize the Notifications 🚀️

-- EXAMPLE --

app_name = "YourText" - Define your NotificationsName

task = "YourTask" - Define your Task

message = "YourMessage" - Define your PopUp Message

toast = "True Or False" - Smaller Notifications Or Bigger

timeout, timesleep = "int" - Interval

# ⛏️ Installations

```
pip install requirements.txt
```
